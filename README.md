# Starter Vue

> Description: Opinionated Vite starter template using [UnoCSS](https://unocss.com) and [shadcn-vue](https://www.shadcn-vue.com), Inspired by [Vitesse](https://github.com/antfu-collective/vitesse).

## Checklist

When you use this template, try follow the checklist to update your info properly

- [ ] Change the package name in `package.json`
- [ ] Change the description in `package.json`
- [ ] Change the author name in `LICENSE`
- [ ] Change the title in `composables/use-unhead.ts`
- [ ] Change the favicon in `public`
- [ ] Replace the `jinghaihan/starter-vue` globally
